В своих проектах я использую Java, который отвечает всем принципам:
1) Использование generic для классов.
2) В классе функции могут активно взаимодействовать, а объединять их можно
например композицией нескольких классов в одном.
3) Класс может наследовать другой класс и множество интерфейсов.
4) Java поддерживает полиморфизм.
5) Можно сделать абстрактный класс, где лишь некоторые функции реализованны.
