В данном уроке необходимо было привести примеры двух типов наследования.
Для льготного наследования был создан класс анализа кода, который будет содержать общие константы с
популярными кодами ответа для остальных классов и стандартное поведение метода, которое можно шаблонно использовать,
дописывая свои условия анализа. С моей точки зрения пример все еще не самый яркий, было бы хорошо например добавить
поле с определенными правилами и метод, который бы выполнял его.
Для наследование реализации был выбран пример фильтрации текста. Представлен класс, который фильтрует "плохие" слова
и фильтр для детских книжек, который наследуется от фильтрации плохих слов и расширяет его с использованием и родительского метода.
