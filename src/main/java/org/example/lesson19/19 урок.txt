Как пример ограничения можно взять класс "Микрофон". У микрофона есть возможность принимать звук и ему нужна определенная мощность
для питания, при этом наследник микрофона "Петличка" может принимать только определенную громкость звука и работает с определенной мощностью.
Если мы возьмем класс "Студийный микрофон", то условия для него будут другими.

Примером расширения может быть класс Телефон и его наследник Смартфон. Смартвон предоставляет множество функций,
которые уже не будут применимы к классу телефон.
